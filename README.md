# Alan Viana - Portfólio

Este é o repositório do meu portfólio pessoal, construído para reunir meus projetos, habilidades e experiências como desenvolvedor front-end. O visual foi inspirado nas interfaces clássicas do Windows 95/98, misturando uma estética retrô com tecnologias modernas de desenvolvimento web.

## Tecnologias utilizadas

- React + TypeScript
- Tailwind CSS
- Framer Motion (para as animações)
- Vite

## Funcionalidades

- Interface nostálgica estilo Win95 com tema claro/escuro
- Layout totalmente responsivo
- Formulário de contato integrado e funcional
- Download direto de currículo em PDF

## Como rodar localmente

Caso queira clonar o projeto e rodar na sua máquina:

```bash
git clone https://github.com/Alan-Viana/Portfolio.git
cd Portfolio
npm install
npm run dev
```
